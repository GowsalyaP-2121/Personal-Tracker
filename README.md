# Personal Tracker

Personal Tracker is a lightweight planner web app for organizing daily tasks in a clean, modern interface.

## Features

- Task planning and day-to-day tracking
- Stylish login and dashboard-style UI
- Single-file frontend project (easy to run anywhere)

## Project Structure

- `index.html` — Main application file (HTML, CSS, and JavaScript)

## Getting Started

1. Clone this repository:

	```bash
	git clone https://github.com/GowsalyaP-2121/Personal-Tracker.git
	```

2. Open the project folder:

	```bash
	cd Personal-Tracker
	```

3. Run the app by opening `index.html` in your browser.

## How to Use

1. Open the app and sign in (or create an account if needed).
2. Add your tasks or plans for the day.
3. Update task status as you complete items.
4. Review your planner regularly to stay on track.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
